Freely contribute, add new companies and update the list (github login).

Info

On joblist.city you can search the available jobs from a list of companies hirring.

This list of companies hirring, is available on this site, and in open-authoring mode. This means that anyone, with a free Github account, can contribute to this list.

Feel free to add a new company, edit the informations about existing ones, or discuss the improvements to these sites.

Note: to submit a company for review (before it being added to the database), you will need to mark your changes as "in review", on the Workflow ("review") page (by dragging the card). Only an admin can publish it when it is in the "ready" state.
